🚀 Monitor de Preços - Bem Vindos
Este é um sistema de monitoramento de produtos inteligente, focado em ajudar o usuário a encontrar as melhores ofertas e ser alertado sobre quedas de preço em tempo real.

📝 Sobre o Projeto
O projeto visa facilitar o acompanhamento de preços de produtos em diversas lojas. O sistema rastreia o histórico, detecta quedas significativas e notifica o usuário, otimizando a economia nas compras online.

🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

Front-end: React.js, TypeScript e Styled-Components.

Gerenciamento de Estado: React Hooks (useState).

Ícones: react-icons.

Design: Responsivo (Mobile-first). (por enquanto)

🏗️ Estrutura do Projeto (Fase 1)

/src
  ├── components/      # Componentes reutilizáveis (Layout, Cards, TabBar)
  ├── Dashboard/       # Componentes específicos da tela inicial
  ├── data/            # Mock de dados (produtos.json)
  └── App.tsx          # Roteamento e orquestração do layout
  🎨 Layout & Design
O sistema utiliza uma abordagem Mobile-first, focada na experiência de usuário (UX) com:

Dashboard Flutuante: Sobreposição elegante entre o Header e o conteúdo.

Navegação Nativa: TabBar fixa com botão de ação (FAB) centralizado.

Temas Coesos: Paleta de cores baseada em tons de azul profundo e fundo neutro para alto contraste.
